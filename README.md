# hci

3D map of casualties in Palestine and Israel since 2000. Made using
Vue and Deck.gl, written in TypeScript.
Made for the [Data Visualization](https://www.unive.it/data/insegnamento/521942)
exam at Ca' Foscari University of Venice.

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```
